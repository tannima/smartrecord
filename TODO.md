# SmartRecord 迭代计划

## v3.0 待开发功能

### 1. 优化语音转文字效果

**目标：** 提升语音识别准确率和速度

**候选方案：**
- [ ] 使用 OpenAI Whisper API（本地/云端）
  - 优点：识别准确率更高，支持多语言，离线可用
  - 缺点：需要API费用或本地计算资源
- [ ] 使用阿里云语音识别
  - 优点：对中文支持更好，速度快
  - 缺点：需要注册账号，有API费用
- [ ] 使用讯飞语音识别
  - 优点：中文识别准确率高
  - 缺点：需要注册账号
- [ ] 集成多个API，支持用户选择

**技术要点：**
- 保持统一的接口，方便切换不同的识别引擎
- 添加配置文件，让用户可以选择使用哪个API
- 实现识别结果缓存机制

### 2. 升级记录存储方式

**目标：** 更便于查看和管理记录

**候选方案：**
- [ ] **集成到macOS备忘录**
  - 使用 AppleScript 直接写入备忘录
  - 可以按日期/标签自动分类
  - 支持iCloud同步，多设备访问
- [ ] **使用Markdown格式**
  - 更好的可读性和组织结构
  - 支持标题、列表、标签
  - 可以用任何Markdown编辑器打开
- [ ] **使用SQLite数据库**
  - 结构化存储
  - 支持标签、分类、搜索
  - 可以开发配套的查看工具
- [ ] **集成到Notion/Obsidian**
  - 通过API写入
  - 更强大的知识管理功能

**技术要点：**
- 保持向后兼容，支持从txt导入
- 添加标签功能
- 实现全文搜索
- 支持导出为多种格式

## 当前版本 v2.0 功能清单

✅ Command+Shift+R 快捷键启动
✅ 智能录音（自动检测语音开始/结束）
✅ E键手动结束录音
✅ ESC键取消录音
✅ 按日期自动保存到txt文件
✅ 自动关闭终端窗口
✅ 3秒停顿检测
✅ 2分钟最长录音时间
